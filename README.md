# \# 🎬 Netflix Insights — SQL + Python + Power BI

# 

# An end-to-end data analytics project exploring Netflix's content library using SQL for data querying, Python for data cleaning and analysis, and Power BI for interactive visualization.

# 

# \---

# 

# \## 📌 Project Overview

# 

# This project dives into Netflix's dataset to uncover trends in content type, genres, release patterns, ratings, and country-wise distribution. The goal is to transform raw data into actionable insights through a full analytics pipeline.

# 

# \---

# 

# \## 🛠️ Tech Stack

# 

# | Tool      | Purpose                              |

# |-----------|--------------------------------------|

# | SQL       | Data exploration and querying        |

# | Python    | Data cleaning, EDA, and analysis     |

# | Power BI  | Interactive dashboards and reporting |

# | Pandas    | Data manipulation                    |

# | Matplotlib / Seaborn | Data visualization (Python) |

# 

# \---

# 

# \## 📂 Project Structure

# 

# ```

# netflix-insights-sql-python-powerbi/

# │

# ├── python/

# └── README.md

# 

# \---

# 

# \## 🔍 Key Analysis \& Insights

# 

# \- 📊 \*\*Content Type Split\*\* — Breakdown of Movies vs TV Shows on Netflix

# \- 🌍 \*\*Top Countries\*\* — Which countries produce the most Netflix content

# \- 📅 \*\*Release Trends\*\* — How Netflix content additions have grown over the years

# \- 🎭 \*\*Popular Genres\*\* — Most common genres across the platform

# \- 🔞 \*\*Ratings Distribution\*\* — Content breakdown by age rating (TV-MA, PG, etc.)

# \- ⏱️ \*\*Movie Duration\*\* — Average movie length and TV show season counts

# \## 🚀 Getting Started

# 

# \### 1. Clone the repository

# ```bash

# git clone https://github.com/aryangiri12/netflix-insights-sql-python-powerbi.git

# cd netflix-insights-sql-python-powerbi

# ```

# 

# \### 2. Install Python dependencies

# ```bash

# pip install pandas matplotlib seaborn jupyter

# ```

# 

# \### 3. Run the EDA notebook

# ```bash

# jupyter notebook python/eda.ipynb

# ```

# 

# \### 4. Run SQL queries

# Open `sql/netflix\_queries.sql` in any SQL client (MySQL Workbench, DBeaver, etc.) and run against the dataset.

# 

# \### 5. Open Power BI Dashboard

# Open `powerbi/netflix\_dashboard.pbix` in Power BI Desktop.
